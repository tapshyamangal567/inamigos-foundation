# Contribution Guidelines

Thank you for your interest in contributing to the **InAmigos Foundation Website** project! 🎉

We welcome contributions that improve the website design, responsiveness, accessibility, performance, and overall user experience.

---

## 🚀 How To Contribute

### 1. Fork The Repository

Click the **Fork** button on the top-right corner of the repository page.

---

### 2. Clone Your Fork

```bash
git clone https://github.com/your-username/inamigos-foundation.git
```

---

### 3. Create A New Branch

```bash
git checkout -b feature-name
```

Example:

```bash
git checkout -b improve-hero-section
```

---

### 4. Make Your Changes

You can contribute by improving:

- UI/UX Design
- Responsive Layout
- Accessibility
- Animations
- Code Structure
- Performance Optimization
- Documentation

---

### 5. Commit Your Changes

```bash
git add .
git commit -m "Improved hero section design"
```

---

### 6. Push Changes

```bash
git push origin feature-name
```

---

### 7. Create A Pull Request

Submit a Pull Request with:
- Proper description
- Screenshots (if UI changes)
- Clear explanation of improvements

---

## 📌 Contribution Rules

- Keep the design clean and responsive
- Follow proper folder structure
- Use meaningful commit messages
- Avoid unnecessary dependencies
- Ensure code readability

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 🌟 Ways To Contribute

You can help by:

- Fixing bugs
- Improving responsiveness
- Enhancing UI design
- Adding animations
- Improving accessibility
- Writing documentation

---

## 🤝 Code Of Conduct

Please be respectful and maintain a positive environment for all contributors.

---

## 💡 Need Help?

If you have suggestions or ideas, feel free to open an issue or discussion.

Happy Contributing 🚀
