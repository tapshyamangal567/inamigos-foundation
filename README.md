# InAmigos Foundation Website

Live Website: https://inamigos-foundation-seven.vercel.app

A modern and responsive NGO website created for **InAmigos Foundation** to showcase its mission, projects, impact, and community initiatives.

---

## 🌍 About The Project

This website represents the vision and activities of **InAmigos Foundation**, a non-profit organization working towards:

- Education
- Women Empowerment
- Environmental Sustainability
- Animal Welfare
- Community Development

The website is designed with a clean and minimal NGO-style interface inspired by modern social impact platforms.

---

## ✨ Features

- Responsive NGO Landing Page
- Modern Hero Section
- About Us Section
- Projects Section
- Impact Section
- Journey Section
- Contact Section
- Social Media Integration
- Smooth Hover Effects & Animations
- Clean UI/UX Design
